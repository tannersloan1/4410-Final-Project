# Learning Management System

A web-based application for a learning environment which provides student, teacher, and admin functionalities.


# Features

### Student

* Register and access classes from dashboard
* Take quizzes assigned from the teachers in those classes
* Recieve instant feedback for all auto-graded questions
* Export data for all classes as a pdf in a report card style
### Teacher

* Manage quizzes and questions for specific classes
* Track student and class progress/scores
* Export data for all classes as CSV
### Admin

* View site-wide analytics
* Manage and create classes
* Control teacher account setup for more secure accounts
* Access activty logs throughout the site with recent 50 on dashboard and more in depth in downloadable CSV


# Tech Stack

Backend: PHP

Database: MySQL (phpMyAdmin/XAMPP)

Frontend: HTML, CSS, JS

Server: Apache

# Setup Instructions

1. Install XAMPP
2. Place project files in the web root (C:\xampp\htdocs\lms)
3. Insert your password if needed in db.php
4. Start Apache and MySQL in the XAMPP Control Panel
5. Import the 'lms.sql' database in phpMyAdmin
6. Go to http://localhost/lms/ to launch
